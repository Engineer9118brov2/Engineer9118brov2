### What I'm building

**[Fable](https://github.com/Engineer9118brov2/Fable)** — a native iOS · macOS · watchOS AI assistant built on top of [OpenClaw](https://github.com/openclaw-ai/openclaw).

- Multi-agent chat with a **conductor pattern** — Scribe orchestrates cloud sub-agents (GLM, MiniMax, Claude)
- **40+ built-in tools** — Calendar, Contacts, Files, Web search, GitHub, iMessage, Hyperliquid, and more
- Long-term memory, live calendar context, and macOS system integration
- Runs privately by default — local Ollama models + optional cloud
